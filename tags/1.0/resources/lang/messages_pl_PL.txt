About_text=Copyright (c) 2007, Locify Ltd. All Rights Reserved\n\nSearch for more at: http://www.locify.com\n\nThis is freeware.
What_is_locify_for=Do czego służy Locify?
What_is_locify_for_text=Locify is a mobile and web application for location-enabled web services. The main content of Locify is services. Services specify functionality and are provided via open API. This means that you can write your own service which requires some location-oriented data and you don't have to solve mobile application programming problems.\n\nSome examples of possible Locify usage: Find nearest KFC restaurants, Find my friends nearby, Tag this place and send it with description to web, Show my position on on-line map, Navigate me to selected geocache, Show nearby Flickr photos in external browser etc.
How_to_add_service=How to add a service?
How_to_add_service_text=You can add a service from the list of verified services on our server. There is a quick link on the main screen or you can use 'Options->Actions->Add->Add service'.\n\nIf a service is not listed in the Locify service list, you can add it manually with the URL. You can do that at 'Options->Actions->Add->Add Manually'.\n\nYou can also add a service to your account via the web at http://www.locify.com. After you synchronize, the service will appear in your mobile phone.\n\nAfter finding the service you want to add, some info about it will be shown. After confirmation, the service is installed and accessible from main screen.
How_to_connect_gps=How to connect GPS?
How_to_connect_gps_text=You can connect the program to a GPS device in multiple ways. By selecting 'Bluetooth GPS' you can connect to a nearby Bluetooth GPS module. Make sure the device is on and Bluetooth is on.\n\nBy selecting 'Internal GPS' you can connect using Location API. This API is used for connecting with internal GPS in some phones like Nokia N95.\n\nUse 'PDA Porter' to connect GPS on PDA's. Read more at http://www.locify.com/help/windows-mobile If you have an external GPS Device like Garmin, 'Without GPS' is the mode for you. You will have to enter coordinates manually, but you can still use Locify.
What_is_login_for=What is login for?
What_is_login_for_text=If you select 'Options->Actions->Login', you can log in to your Locify account. After login you can synchronize all the data in your phone with the Locify server. Some services can also use your Locify login for their authentication.
What_is_synchronized=What is synchronized?
What_is_synchronized_text=Everything. All data is synchronized, so you can switch phones and after synchronization you will see the Locify client the same as before. You can also manage your services on the web and these changes will be visible on your phone after synchronization.
How_to_manage_services=How to manage items on main screen?
How_to_manage_services_text=You have various options in managing items. You can add new services, shortcuts, delete them, move their position on the main screen etc. Everything is done by selecting Service in the main screen and then 'Options->Actions->Manage Item'.
Where_is_service_settings=Where are service settings?
Where_is_service_settings_text=Every service can have its own settings. You can customize the service via these settings. Settings are available at 'Options->Actions->Manage Item->Settings'. The settings option will be shown only if there are any settings.
How_can_i_save_my_location=How can I save my location?
How_can_i_save_my_location_text=You need a service Locations to do that. Install service and select first item in menu - Save locally. Now you can save current or manually entered locations. If you want to send your offline locations to Locify server, just synchronize.
How_can_i_navigate_to_own_coordinates=How can I navigate to own coordinates?
How_can_i_navigate_to_own_coordinates_text=Navigation is mostly used when you receive waypoint from some service. Howewer you can navigate to manually entered coordinates as well. Start service Locations, select Save Locally and create waypoint with your manually entered coordinates. Then go to 'Options->Tools->Filesystem', select your saved location and click Navigate.
